## 3. Persyaratan Sistem

Pastikan Anda telah menginstal komponen berikut di sistem Anda:

* **Flutter SDK:** Versi Stable (Minimal 3.19.x atau terbaru).
* **Git:** Untuk kloning repositori.
* **Android SDK/NDK:** Jika ingin menjalankan pada perangkat fisik/emulator Android.
